## 1. 제품 현황 진단 (As-Is)

### 현재 Sporton이 가진 것
| 영역 | 현황 | 평가 |
|------|------|------|
| 핵심 기능 | 실시간 라이브스코어 (ESPN API) | ✅ 강점 |
| 배당률 | The Odds API 통합 | ✅ 차별점 |
| 커뮤니티 | 글쓰기/댓글 (Firebase) | 🟡 기초 수준 |
| UI/UX | 다크모드, 반응형, 4컬럼 그리드 | ✅ 양호 |
| 모바일 | 기본 반응형 지원 | 🟡 최적화 필요 |
| 알림 | 없음 | 🔴 미존재 |
| 개인화 | 없음 | 🔴 미존재 |
| 소셜 기능 | 없음 | 🔴 미존재 |
| SEO | 기본 메타 태그만 | 🔴 취약 |
| 분석 | 없음 | 🔴 미존재 |

### 핵심 문제 진단 (5 WHY)
```
WHY 동시접속 1,000명이 안 되는가?
└─ WHY 1: 유저가 돌아올 이유가 없다
   └─ WHY 2: 알림/훅이 없어서 경기 시작을 모른다
      └─ WHY 3: 푸시 알림, 즐겨찾기 팀 기능이 없다
         └─ 해결책: 팀 팔로우 + 경기 시작 알림
└─ WHY 2: 친구와 함께 즐길 수 없다
   └─ WHY 3: 공유 기능, 예측 내기, 실시간 채팅이 없다
      └─ 해결책: 소셜 예측, 경기별 라이브 채팅
└─ WHY 3: 검색으로 찾아올 수 없다
   └─ WHY 4: SEO가 거의 없어 구글/네이버 노출 안됨
      └─ 해결책: 경기별 정적 페이지, 구조화 데이터
└─ WHY 4: 한 번 와서 할 게 없다
   └─ WHY 5: 배당률 + 라이브스코어만으로는 체류시간 짧음
      └─ 해결책: 예측 게임, 커뮤니티 강화, 통계 분석
```

## 2. 제품 비전 & 전략 (To-Be)

### 제품 비전
> **"한국 스포츠 팬이라면 경기 중 반드시 켜놓는 탭 — Sporton"**

경기 시작 5분 전 알림 → Sporton 접속 → 실시간 스코어 확인 → 커뮤니티에서 토론 → 예측 적중 → 친구에게 공유 → 친구도 가입

### 포지셔닝 전략
```
현재 경쟁 지형:
├─ 네이버 스포츠: 대형 포털, 개인화 없음, 커뮤니티 약함
├─ 다음 스포츠: 레거시 UI, 실시간성 낮음
├─ 엑스스코어: 배당률 특화, UX 구식
└─ SPORTON 포지셔닝:
   → 실시간 라이브스코어 + 배당률 + 커뮤니티 통합
   → 20~35세 한국 스포츠 팬 타겟
   → 모바일 퍼스트, 경량 UX
   → 커뮤니티 기반 예측 게임
```

### 전략 방향 (3가지 기둥)
| 기둥 | 방향 | 기대 효과 |
|------|------|-----------|
| **리텐션** | 알림 + 팀 팔로우로 재방문율 높이기 | D7 리텐션 30%→55% |
| **바이럴** | 예측 공유로 유기적 신규 유저 획득 | K-factor 0.3→0.7 |
| **인게이지먼트** | 경기 중 실시간 채팅으로 체류시간 증가 | 세션당 체류 3분→12분 |

## 3. North Star Metric

### 선정 과정
| 후보 지표 | 비즈니스 기여 | 측정 용이성 | 팀 영향력 | 최종 |
|-----------|--------------|------------|----------|------|
| DAU | 중 | 쉬움 | 높음 | ❌ |
| **경기당 활성 체류 유저수** | **최고** | **중간** | **최고** | **✅ NSM** |
| 총 예측 제출수 | 중 | 쉬움 | 중간 | ❌ |
| 커뮤니티 댓글수 | 낮음 | 쉬움 | 중간 | ❌ |

### North Star Metric 정의
```
NSM: "경기 중 10분 이상 활성 체류한 동시접속 유저 수"
(Live Active Concurrent Users, LACU)

현재 추정치: ~50명 (피크)
6개월 목표: 1,000명 (피크)
```

### Input Metrics (NSM을 움직이는 레버)
```
LACU 1,000명 달성을 위한 Input Metrics:
├─ 획득 (Acquisition)
│  ├─ 신규 유저 주간 가입수 (목표: 500명/주)
│  ├─ 오가닉 검색 유입 (목표: DAU의 40%)
│  └─ 공유 바이럴 유입 (목표: DAU의 20%)
├─ 활성화 (Activation)
│  ├─ 가입 후 경기 1개 이상 시청률 (목표: 70%)
│  └─ 팀 팔로우 완료율 (목표: 가입자의 50%)
├─ 리텐션 (Retention)
│  ├─ D7 리텐션 (목표: 55%)
│  ├─ 경기 알림 → 접속 전환율 (목표: 40%)
│  └─ 주간 3회 이상 방문 비율 (목표: 35%)
└─ 수익화 (Referral)
   ├─ 예측 공유 클릭률 (목표: 20%)
   └─ 친구 초대 전환율 (목표: 10%)
```

## 5. 유저 페르소나 & 고객 여정

### 페르소나 1: 박민준 (핵심 타겟) — "배팅 좀 아는 축구 팬"
```
나이: 28세 | 직업: 직장인 | 스포츠: 해외 축구 (프리미어리그)
행동: 퇴근 후 경기 챙겨봄, 배당률 비교 즐겨함
페인포인트:
  - 배당률 보려면 토토 사이트 들어가야 함 (불편)
  - 커뮤니티와 라이브스코어가 따로 분리됨
  - 경기 시작 알림 없어서 놓치는 경우 많음
Sporton이 줄 가치:
  → 라이브스코어 + 배당률 한 화면
  → 예측 적중 시 친구에게 자랑
  → 경기 시작 알림
```

### 페르소나 2: 이지은 — "NBA 팬, 라이트 유저"
```
나이: 24세 | 직업: 대학원생 | 스포츠: NBA
행동: 새벽 NBA 결과 확인, 커뮤니티 눈팅 주도
페인포인트:
  - 새벽에 경기 결과만 빠르게 보고 싶음
  - 앱 설치 싫음, 모바일 웹으로만 사용
  - 한국어 NBA 커뮤니티 마땅한 곳 없음
Sporton이 줄 가치:
  → 빠른 모바일 결과 확인
  → 한국어 NBA 커뮤니티
  → PWA 설치 (앱 없이 홈화면에)
```

### 페르소나 3: 김태호 — "스포츠 인플루언서"
```
나이: 32세 | 직업: 유튜버/스트리머
행동: 경기 예측 콘텐츠 만들고 커뮤니티 관리
페인포인트:
  - 팔로워들과 실시간 예측 배틀 할 플랫폼 없음
  - 예측 적중률 기록/공유 도구 없음
Sporton이 줄 가치:
  → 예측 리더보드 (본인 포지셔닝)
  → 커뮤니티 글 작성 및 팔로워 확보
  → 공유 가능한 예측 카드
```

### 고객 여정 맵 (민준 기준)
```
인지 → 가입 → 활성화 → 리텐션 → 추천

인지:
  - 구글에서 "오늘 맨시티 경기 배당률" 검색
  - 트위터/인스타에서 예측 카드 공유 글 봄
  - 친구가 예측 적중 결과 공유

가입 (목표: 60초 내 완료):
  - 소셜 로그인 (카카오/구글) → 팀 선택 → 완료
  - 즉시 다음 경기 알림 설정 유도

첫 경험 (Aha Moment 목표: 3분 내):
  - 맨시티 경기 카드 클릭 → 라이브 스코어 + 배당률 동시 확인
  - "오, 이거 한 화면에 다 있네" → AHA!

리텐션 훅:
  - D1: 경기 결과 푸시 알림 수신
  - D3: 팔로우 팀 다음 경기 알림
  - D7: 주간 예측 적중률 리포트

추천:
  - 예측 적중 → 공유 카드 생성 → 카카오/인스타 공유
  - 친구 가입 → 두 사람 모두 예측 포인트 보너스
```

## 6. 핵심 성장 루프

### 루프 1: 예측 바이럴 루프 (Primary Growth Loop)
```
[경기 예측 참여]
      ↓
[결과 발표 - 적중/실패]
      ↓
[공유 가능한 예측 카드 자동 생성]
      ↓
[SNS/카카오 공유]
      ↓
[친구가 링크 클릭 → Sporton 방문]
      ↓
[신규 가입 & 예측 참여]
      ↓
[경기 예측 참여] (루프 반복)

측정 지표: 공유율, 공유→방문율, 방문→가입율
목표 K-factor: 0.7 (신규 유저 1명이 0.7명 데려옴)
```

### 루프 2: 알림 리텐션 루프 (Retention Loop)
```
[팀 팔로우 설정]
      ↓
[경기 시작 30분 전 알림]
      ↓
[Sporton 접속]
      ↓
[경기 중 실시간 체류]
      ↓
[커뮤니티 댓글 / 예측 참여]
      ↓
[다음 경기도 팔로우] (루프 강화)

측정 지표: 알림→접속 CTR, 경기 중 체류시간
목표: 알림 CTR 40%, 체류 12분
```

### 루프 3: SEO 오가닉 루프 (Acquisition Loop)
```
[경기별 SEO 최적화 페이지 생성]
      ↓
[구글/네이버 검색 상위 노출]
      ↓
["오늘 프리미어리그 경기" 검색 → Sporton]
      ↓
[라이브스코어 + 커뮤니티 경험]
      ↓
[가입 & 팀 팔로우]
      ↓
[유저 생성 콘텐츠 (댓글/예측)]
      ↓
[검색 엔진 품질 신호 상승] (루프 강화)
```
