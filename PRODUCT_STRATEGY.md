## 1. 현황 진단

### 2026-03-18 기준 제품 상태

| 영역 | 이전 상태 (3/7) | 현재 상태 (3/18) | 평가 |
|------|----------------|-----------------|------|
| 핵심 라이브스코어 | ✅ ESPN API | ✅ 동일 + 타임라인 추가 | ✅ 강점 |
| 배당률 | ✅ The Odds API | ✅ 동일 | ✅ 차별점 |
| 팀 팔로우 | 🔴 없음 | ✅ localStorage 기반 구현 | ✅ 완료 |
| 웹 푸시 알림 | 🔴 없음 | ✅ Service Worker 구현 | ✅ 완료 |
| 예측 게임 | 🔴 없음 | ✅ 포인트/리더보드 포함 | ✅ 완료 |
| 실시간 채팅 | 🔴 없음 | ✅ 경기별 채팅 + 전체 채팅 | ✅ 완료 |
| 소셜 로그인 | 🔴 없음 | 🟡 Google만 구현 (카카오 미구현) | 🟡 **GAP** |
| 예측 공유 카드 | 🔴 없음 | ✅ Canvas 기반 구현 | ✅ 완료 |
| 리더보드 | 🔴 없음 | ✅ 주간/전체 구현 | ✅ 완료 |
| SEO | 🔴 기초 메타태그 | ✅ Schema.org + 동적 메타태그 | ✅ 완료 |
| PWA | 🔴 없음 | ✅ manifest + SW 설치 | ✅ 완료 |
| AdSense | 🔴 없음 | ✅ 코드 설치 완료 | 🟡 활성화 필요 |
| 친구 초대 | 🔴 없음 | ✅ 추천 코드 시스템 구현 | ✅ 완료 |
| 관리자 대시보드 | 🔴 없음 | ✅ CCU + 통계 구현 | ✅ 완료 |
| 커뮤니티 | 🟡 기초 | ✅ 포럼 + 댓글 구현 | ✅ 완료 |
| 팀명/선수명 한글화 | 🔴 없음 | ✅ 번역 딕셔너리 구현 | ✅ 완료 |
| 종목 확장 | 🟡 5종목 | 🟡 5종목 유지 | 🟡 중기 과제 |
| 온보딩 최적화 | 🔴 없음 | 🔴 미구현 | 🔴 **GAP** |
| 마케팅 실행 | 🔴 없음 | 🔴 미실행 | 🔴 **즉시 필요** |

### 핵심 진단: 제품은 준비됐다, 이제 성장 엔진을 켜야 한다

```
✅ 제품 완성도: ~80% (P0/P1 기능 대부분 구현)
🔴 Growth Engine: ~10% (실제 유저 획득/유지 실행 안됨)
🔴 Revenue: ~5% (AdSense 코드만 있고 수익 미발생)
🟡 Community: ~30% (기능은 있지만 시드 콘텐츠/유저 없음)
```

**결론**: 지금 필요한 것은 "더 많은 기능"이 아니라 **"유저를 데려오고 붙잡는 것"**

## 3. 비즈니스 임팩트 분석

### 3대 핵심 성장 레버

#### 레버 1: 한국어 UX 품질 (기반 강화)

```
현재 완료: 팀명/선수명 전체 한글화 구현 ✅

기존 문제:
- "Manchester City vs Arsenal" → 영어로 표시
- "L. Messi", "H. Kane" → 알아보기 어려운 약어
- 한국 유저 이탈 원인 중 하나

한글화 후 효과:
- "맨체스터 시티 vs 아스날" → 즉각 인식
- "메시", "케인" → 친숙한 표기
- 신뢰도 상승 + 체류시간 증가

커버된 팀/리그:
✅ EPL 20팀 | ✅ 라리가 20팀 | ✅ 분데스리가 18팀
✅ 세리에A 20팀 | ✅ 리그앙 18팀 | ✅ NBA 30팀
✅ MLB 30팀 | ✅ NFL 32팀 | ✅ NHL 32팀
✅ 유명 선수 50명+ 한국어 표기

예상 임팩트: 한국 유저 첫 방문 이탈률 -20%, 체류시간 +15%
```

#### 레버 2: GTM 실행 (트래픽 0→유입)

```
현재 문제: 훌륭한 제품이지만 아무도 모른다

채널별 예상 유입 (월):
├─ 디시인사이드 스포츠 갤러리: 경기 예측 글 10개/주 → 500명/월
├─ 네이버 카페 (EPL/NBA): 공략 글 5개/주 → 300명/월
├─ 트위터/X: 경기 예측 콘텐츠 → 200명/월
├─ SEO 오가닉 (구글): 3개월 후 → 1,000명/월
└─ 예측 카드 바이럴: K-factor 0.3 달성 시 → 자연 성장

총 예상: 런칭 1달 후 DAU 100명, 3달 후 DAU 500명 가능
```

#### 레버 3: 수익화 (지속 가능성)

```
현재 수익: 0원

AdSense 수익 예측 (CPM 기준):
├─ DAU 100명, 페이지뷰 500 → ~$1/일 ($30/월)
├─ DAU 500명, 페이지뷰 2,500 → ~$5/일 ($150/월)
├─ DAU 2,000명, 페이지뷰 10,000 → ~$20/일 ($600/월)
└─ DAU 5,000명, 페이지뷰 25,000 → ~$50/일 ($1,500/월)

목표: 3개월 내 DAU 500명 달성 → 서버비 + 운영비 자급자족
```

## 7. 수익화 전략 업데이트

### 3단계 수익화 모델

#### Stage 1: AdSense (즉시~3개월)
```
현재 상태: 코드 설치 완료, Publisher ID만 설정하면 즉시 수익

광고 위치 최적화:
  ├─ 경기 리스트 사이사이 (인피드 광고) — CTR 높음
  ├─ 경기 상세 하단 (콘텐츠 광고)
  ├─ 채팅 영역 상단 (사이드바 광고)
  └─ 예측 결과 후 (보상형 광고 느낌)

예상 수익:
  DAU 100명 → $30/월
  DAU 500명 → $150/월
  DAU 2,000명 → $600/월
```

#### Stage 2: Sporton Plus (3~6개월)
```
구독 플랜 ($2/월 또는 $15/년):
  - 광고 제거
  - 고급 배당률 분석 (Line Movement)
  - 예측 상세 통계
  - 프리미엄 배지 + 닉네임 컬러
```
