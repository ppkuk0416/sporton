# pm-toolkit

PM utility skills: resume review, document proofreading, and privacy policy generation.

## Overview

A collection of general-purpose skills for PMs covering career documents, professional writing, and legal templates. These utilities are useful across all stages of a PM's work.

## Install

```bash
claude plugin install pm-toolkit
```

## Skills

| Skill | Description |
|-------|-------------|
| `review-resume` | PM resume review using XYZ+S formula and 10 best practices |
| `grammar-check` | Document editing for grammar, clarity, flow, and professional tone |
| `privacy-policy` | Privacy policy generation covering GDPR, CCPA, and COPPA |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-toolkit:review-resume` | Review and rewrite a PM resume with ATS optimization |
| `/pm-toolkit:proofread` | Proofread and improve any document for a specific audience |
| `/pm-toolkit:privacy-policy` | Generate a privacy policy for a digital product |

## Example Usage

```
/pm-toolkit:review-resume [paste resume] target job: Senior PM at Stripe

/pm-toolkit:proofread [paste PRD] audience: engineering

/pm-toolkit:privacy-policy TaskFlow app by Acme Inc collects email, analytics, payments
```
