# Proofread

Edit a document for grammar, clarity, logic, and professional quality.

## Step 1: Identify the Document and Audience

If not provided, ask:
- What type of document is this? (PRD / email / presentation / stakeholder update / etc.)
- Who is the intended audience? (Executive / Engineering / Customer / Investor)
- What is the goal of the document? (Inform / persuade / request / update)

## Step 2: Run the Edit

Activate the **grammar-check** skill.

Apply all five layers: grammar, clarity, flow, tone, and structure.

## Step 3: Calibrate for Audience

Adjust tone and complexity for the stated audience:
- Executive: Trim to essentials, lead with the recommendation or insight
- Engineering: Add precision, remove ambiguity, show reasoning
- Customer: Simplify language, add warmth, focus on benefits not features
- Investor: Strengthen data points, remove hedging language

## Step 4: Deliver Edits

Provide:
1. **Issues found**: Numbered list of specific problems
2. **Edit-level**: Is this a light polish or a significant rewrite? Be transparent.
3. **Rewritten document**: Full version with all edits applied
4. **Quality scores**: Before/after on clarity, professionalism, conciseness, flow

Ask: "Would you like any section rewritten in a different tone or style?"
