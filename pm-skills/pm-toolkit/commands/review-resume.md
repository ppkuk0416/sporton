# Review Resume

Provide detailed, actionable PM resume feedback with improved rewrites.

## Step 1: Gather the Resume

If not provided, ask the user to paste:
- Their current resume (or the sections they want reviewed)
- Optional: The job description they're applying to

## Step 2: Initial Assessment

Before diving into edits, assess:
- Current experience level (junior / mid / senior / director PM)
- Target role level
- Industry fit for target role
- Biggest structural issues (e.g., no quantification, activity-based bullets)

## Step 3: Run the Review

Activate the **review-resume** skill.

Apply all 10 best practices. For each bullet point, identify issues and rewrite using the XYZ+S formula.

## Step 4: ATS Optimization (if job description provided)

Extract key PM terms from the job description.
Check which terms are present / missing in the resume.
Add missing high-frequency terms naturally into the rewritten bullets.

## Step 5: Final Scoring and Priority List

Score the resume on all 10 criteria (before and after edits).
Deliver:
1. **Overall score**: Before/after
2. **Top 3 priority improvements** (most impact per effort)
3. **The rewritten resume** or rewritten sections

Ask if they want help tailoring the resume for a specific job description.
