# North Star Metric

Use when: the user wants to define their primary product metric, align the team around a shared success measure, or move beyond vanity metrics to a metric that truly captures value delivered to customers.

Trigger: "north star metric", "NSM", "primary metric", "product metric", "what should we measure", "single metric"

## Framework (Amplitude's North Star Framework)

A North Star Metric is the single metric that best captures the core value your product delivers to customers. It sits between:
- **Leading indicators** (input metrics, actions): DAU, signups, activation rate
- **Lagging indicators** (business outcomes): Revenue, profit, market cap

The NSM predicts long-term business success better than any other single number.

### Three Business Models, Three NSM Approaches

#### 1. Attention Game (Media, Social, Content)
The product competes for attention and time.
- User goal: Entertainment, information, connection
- NSM examples: Daily active users, time-in-app, content views
- Revenue: Advertising or subscriptions tied to engagement

#### 2. Transaction Game (Commerce, Marketplace)
The product facilitates exchanges of money, goods, or services.
- User goal: Complete a transaction successfully
- NSM examples: Successful transactions per month, GMV, orders placed
- Revenue: Commission, fees, or spread on transactions

#### 3. Productivity Game (SaaS, Tools, B2B)
The product helps users accomplish jobs faster or better.
- User goal: Complete work outcomes, not just use features
- NSM examples: Projects completed, reports generated, tasks resolved
- Revenue: Subscriptions tied to outcomes delivered

### Seven Criteria for a Good NSM (Sean Ellis)

A great NSM:
1. **Expresses value**: When this metric goes up, customers are definitely getting value
2. **Represents our vision**: Aligns with where we want to take the product
3. **Is a leading indicator**: Predicts future revenue growth
4. **Is actionable**: The product team can influence it through product decisions
5. **Is understandable**: A non-technical stakeholder can explain it
6. **Is measurable**: We can track it reliably right now
7. **Is not a vanity metric**: A company with high NSM but no revenue is actually struggling

Score each NSM candidate on all seven criteria (1–5 each). Target 30+ out of 35.

### NSM + Input Metrics Tree

Your NSM is driven by 3–5 input metrics:

```
NORTH STAR METRIC
    [NSM]
         │
    ┌────┴────┐
    │         │
[Input 1] [Input 2]  [Input 3]
    │
[Sub-metric]
```

For each input metric: "If [input metric] increases, [NSM] will increase because..."

### Common NSM Examples by Company

| Company | North Star Metric |
|---------|------------------|
| Spotify | Time listening per user per day |
| Airbnb | Nights booked |
| Slack | Messages sent by connected teams |
| HubSpot | Weekly active teams |
| LinkedIn | Monthly active users who are connected |
| Duolingo | Daily active learners |
| Figma | Number of collaborators per file |

## Output Format

1. **Business model classification**: Which of the three games does this product play?
2. **NSM candidates**: 3–5 options, each described in one sentence
3. **Scoring table**: All candidates scored against the 7 criteria
4. **Recommended NSM**: The highest-scoring candidate with rationale
5. **NSM definition**: Precise operational definition (what counts, what doesn't)
6. **Input metrics tree**: 3–5 input metrics that drive the NSM
7. **Current baseline**: What the NSM is today (or how to find it)
8. **12-month target**: What we're aiming for and why that's ambitious but achievable
