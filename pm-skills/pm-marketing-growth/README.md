# pm-marketing-growth

Product marketing and growth skills: north star metrics, positioning, and value proposition copy.

## Overview

This plugin bridges product strategy and marketing execution. It helps PMs define the metric that matters most (NSM), position the product to win in a crowded market, and write compelling copy for every channel.

## Install

```bash
claude plugin install pm-marketing-growth
```

## Skills

| Skill | Description |
|-------|-------------|
| `north-star-metric` | Define the single metric that captures long-term customer value |
| `positioning-ideas` | Generate positioning concepts to differentiate in a crowded market |
| `value-prop-statements` | Write value proposition copy for landing pages, sales, onboarding, and more |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-marketing-growth:north-star` | Define your NSM, input metrics tree, and 12-month target |
| `/pm-marketing-growth:market-product` | Build a full product marketing toolkit with positioning and copy |

## Example Usage

```
/pm-marketing-growth:north-star B2B project management SaaS subscription model

/pm-marketing-growth:market-product AI writing tool for content marketers vs Jasper Copy.ai
```
