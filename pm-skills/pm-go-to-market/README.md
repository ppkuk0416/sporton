# pm-go-to-market

Go-to-market skills: launch strategy, growth loops, ideal customer profiling, and competitive battlecards.

## Overview

This plugin covers the full GTM lifecycle — from defining your ideal customer and designing growth loops to building launch plans and competitive materials. Based on GTM frameworks from Crossing the Chasm, Product-Led Growth, and modern sales methodology.

## Install

```bash
claude plugin install pm-go-to-market
```

## Skills

| Skill | Description |
|-------|-------------|
| `gtm-strategy` | Design a comprehensive GTM strategy with channel plan and launch phases |
| `growth-loops` | Design self-reinforcing growth loops (viral, usage, collaboration, content, paid) |
| `competitive-battlecard` | Build a sales battlecard with objection handling and differentiation |
| `ideal-customer-profile` | Define a rigorous ICP with scoring model and disqualifiers |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-go-to-market:plan-launch` | Full GTM plan for a product or feature launch |
| `/pm-go-to-market:battlecard` | Competitive battlecard for a specific competitor |

## Example Usage

```
/pm-go-to-market:plan-launch new AI writing assistant launch date April 1

/pm-go-to-market:battlecard Notion
```
