# Plan OKRs

Design a complete OKR set for a team or company.

## Step 1: Understand Strategic Context

If not provided, ask:
- What team is this for? (company-wide or specific team)
- What quarter / time period?
- What are the company's top 1–2 priorities right now?
- What did last quarter's OKRs look like? What was the final score?
- Are there any specific outcomes the team must contribute to?

## Step 2: Generate OKR Candidates

Activate the **brainstorm-okrs** skill.

Generate 4–6 Objective candidates. For each, draft 3–4 Key Result options. More is better at this stage — we'll trim in Step 3.

## Step 3: Score and Select

For each Objective candidate, score:
- **Strategic alignment** (1–5): Does it contribute to company priorities?
- **Team ownership** (1–5): Can this team actually move this metric?
- **Ambition** (1–5): Is it a meaningful stretch?
- **Measurability** (1–5): Are the KRs specific and trackable?

Select the top 2–3 Objectives with the highest combined score.

For selected Objectives, refine KRs:
- Add baselines and targets for all KRs
- Assign confidence scores (start at 50% for new OKRs)
- Assign owners

## Step 4: Anti-Pattern Check

Review selected OKRs against common OKR mistakes:
- Any Objectives that are tasks, not directions? → Rewrite
- Any KRs that measure activity, not outcomes? → Rewrite
- Any KRs without a baseline? → Add baseline
- More than 4 KRs per Objective? → Trim to 3

## Step 5: Alignment Map

Show how each team OKR connects to a company OKR. Flag any team OKR that doesn't link to a company priority — these need justification.

## Step 6: Deliver

Final OKR document with:
- 2–3 Objectives
- 3–4 KRs each with baseline, target, confidence, owner
- Alignment map
- Anti-pattern check results
- Suggested weekly check-in cadence
