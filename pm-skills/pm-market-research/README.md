# pm-market-research

User research, competitive analysis, market sizing, and feedback analysis skills.

## Overview

This plugin provides structured frameworks for understanding markets, customers, and competitors. From building credible TAM/SAM/SOM models to extracting themes from customer feedback, it turns raw research into strategic insights.

## Install

```bash
claude plugin install pm-market-research
```

## Skills

| Skill | Description |
|-------|-------------|
| `user-personas` | Create research-grounded behavioral personas (not demographic) |
| `competitor-analysis` | Profile competitors across positioning, product, pricing, and vulnerabilities |
| `market-sizing` | Calculate TAM/SAM/SOM using top-down and bottom-up methods |
| `sentiment-analysis` | Extract themes and priorities from customer feedback |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-market-research:competitive-analysis` | Full competitive landscape with strategic implications |
| `/pm-market-research:analyze-feedback` | Analyze customer reviews, surveys, or support tickets |

## Example Usage

```
/pm-market-research:competitive-analysis our project management tool vs Asana, Linear, Monday

/pm-market-research:analyze-feedback [paste 20 G2 reviews here]
```
