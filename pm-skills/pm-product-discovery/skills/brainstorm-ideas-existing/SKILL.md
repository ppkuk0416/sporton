# Brainstorm Ideas for Existing Products

Use when: the user wants to generate ideas to improve, grow, or expand an existing product. Useful for quarterly planning, growth reviews, and product-led growth initiatives.

Trigger: "brainstorm improvements", "how do we grow the product", "ideas for existing product", "what should we build next"

## Framework

### Layer 1: Core Loop Deepening
Analyze the product's core action loop:
```
Trigger → Action → Variable Reward → Investment
```
For each element, ask:
- How can we reduce friction in this step?
- How can we make the reward more immediate or meaningful?
- How can we increase the stickiness of the investment?

Generate 3 ideas per element.

### Layer 2: Retention Drivers
Identify the top reasons users churn or go dormant. For each reason:
- What feature or experience change would prevent this?
- What notification or re-engagement would bring them back?

### Layer 3: Feature Adjacency Map
List the top 5 tasks users do before/after using your product:
- What do they do right before opening your app?
- What do they do right after?

For each adjacent task: "What if we owned this too?"

### Layer 4: Power User Study
Identify behaviors of your top 10% users (retention, NPS, revenue):
- What do they do in their first week that others don't?
- What features do they use that others ignore?

Turn each power user behavior into a feature or onboarding change for regular users.

### Layer 5: Competitive Gap Analysis
List 3 competitors. For each: what do they do that you don't?
- Which gaps represent real user needs?
- Which gaps are table stakes to close?

## Idea Scoring

For each idea:
- **Impact**: Expected effect on key metric (Low / Med / High)
- **Effort**: Engineering complexity (S / M / L)
- **Confidence**: How sure are we customers want this? (1–5)

Quick wins = High impact + Small effort + High confidence.

## Output Format

Produce 15–20 ideas organized by layer. For each idea:
- **Idea**: [Name]
- **Hypothesis**: If we [do X], then [metric] will improve because [reason]
- **Metric moved**: [Specific KPI]
- **Impact / Effort / Confidence**: H/M/L | S/M/L | 1-5

End with a prioritized top 5 list with recommended next steps.
