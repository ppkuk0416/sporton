# pm-product-discovery

Product discovery skills and workflows based on Teresa Torres's Continuous Discovery framework.

## Overview

This plugin provides structured frameworks for the full discovery cycle: from ideation and opportunity mapping to customer interviews and assumption testing. It operationalizes the Opportunity Solution Tree (OST) methodology and continuous discovery habits.

## Install

```bash
claude plugin install pm-product-discovery
```

## Skills

| Skill | Description |
|-------|-------------|
| `opportunity-solution-tree` | Build an OST linking outcomes → opportunities → solutions → experiments |
| `brainstorm-ideas-new` | Generate ideas for new products using 4 ideation lenses |
| `brainstorm-ideas-existing` | Generate growth and improvement ideas for existing products |
| `interview-script` | Create a structured customer interview guide |
| `identify-assumptions` | Map and prioritize risky assumptions using a 5-category framework |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-product-discovery:discover` | Full discovery cycle from desired outcome to prioritized experiments |
| `/pm-product-discovery:interview` | Generate a tailored customer interview guide |
| `/pm-product-discovery:brainstorm` | Structured ideation for new or existing products |

## Example Usage

```
/pm-product-discovery:discover retention for my B2B SaaS product

/pm-product-discovery:interview churned enterprise customers

/pm-product-discovery:brainstorm existing fitness tracking app
```
