# Value Proposition

Create a complete, validated value proposition for a product.

## Step 1: Get Context

If not provided, ask:
- What product is this for?
- Who is the primary customer segment?
- What alternatives do customers currently use?

## Step 2: Build the Value Proposition Canvas

Activate the **value-proposition** skill.

Map the full customer profile (jobs, pains, gains) and the value map (pain relievers, gain creators). Assess fit across all elements.

## Step 3: Assess Competitive Differentiation

For each pain reliever and gain creator, ask:
- Do competitors offer this? (Yes / Partially / No)
- If yes, do we do it better? How?

Flag elements where we have **exclusive** or **superior** delivery — these are the core of the differentiation story.

## Step 4: Generate Multiple VP Versions

Produce 3 versions of the value proposition for different contexts:

**Elevator pitch** (10 seconds):
"[Product] helps [customer] [achieve outcome] by [key mechanism]."

**Landing page headline** (one bold statement):
Focus on the most emotionally resonant gain or pain relief.

**Sales deck opener** (2–3 sentences):
Problem → Solution → Proof point.

## Step 5: Test for Clarity

Check each version against:
- Does a stranger understand it without context?
- Is the customer the hero (not the product)?
- Does it make a specific, believable promise?
- Does it differentiate from the obvious alternative?

Rate each: 🟢 Strong / 🟡 Needs refinement / 🔴 Rewrite

Deliver all three versions with ratings and suggested improvements.
