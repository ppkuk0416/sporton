# pm-product-strategy

Strategic frameworks for product vision, positioning, business models, and pricing.

## Overview

This plugin encodes proven product strategy frameworks from Marty Cagan (Inspired), Geoffrey Moore (Crossing the Chasm), and Strategyzer (Business Model Canvas / Value Proposition Canvas). It helps PMs build durable strategies grounded in customer value.

## Install

```bash
claude plugin install pm-product-strategy
```

## Skills

| Skill | Description |
|-------|-------------|
| `product-vision` | Craft an inspiring, durable product vision (Geoffrey Moore + Marty Cagan) |
| `value-proposition` | Design a value proposition using the VP Canvas and JTBD |
| `swot-analysis` | Conduct a SWOT + TOWS analysis with strategic options |
| `business-model` | Map the Business Model Canvas and Lean Canvas |
| `pricing-strategy` | Design pricing tiers, models, and value anchoring |

## Commands

| Command | Description |
|---------|-------------|
| `/pm-product-strategy:strategy` | Full product strategy document |
| `/pm-product-strategy:value-proposition` | Value proposition design and testing |
| `/pm-product-strategy:pricing` | Comprehensive pricing strategy |

## Example Usage

```
/pm-product-strategy:strategy my B2B project management tool

/pm-product-strategy:value-proposition TaskFlow for remote engineering teams

/pm-product-strategy:pricing SaaS analytics tool B2B current price $49/month
```
