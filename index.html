Firebase 연결 설정

아래 순서대로 따라하면 5분 안에 커뮤니티가 작동해요!

  1. Firebase 접속
    console.firebase.google.com → 구글 로그인
  2. 프로젝트 만들기
    "프로젝트 추가" → 이름 입력 (예: livescore-pro) → 계속 → 완료
  3. 웹 앱 추가
    프로젝트 홈에서 </> 아이콘 클릭 → 앱 닉네임 입력 → "앱 등록"
  4. 설정 코드 복사
    아래처럼 생긴 코드가 나와요. firebaseConfig = { ... } 안의 내용 전체를 복사해서 아래에 붙여넣으세요
  5. Firestore 활성화
    왼쪽 메뉴 "Firestore Database" → "데이터베이스 만들기" → "테스트 모드" 선택 → 완료
